# 需求规格说明 — 「字卡」恋爱陪伴App

> 版本：v3 | 更新日期：2026-07-03

---

## 项目概述

**字卡**是一款仅个人使用的恋爱陪伴向 PWA 应用。以模拟与另一半对话为核心，辅以陪伴计时、每日记录、书信、生理期追踪等功能。数据全本机存储，不上传服务器。

## 用户画像

- 单人使用，无需账号登录
- 华为手机，Android + 鸿蒙浏览器
- 偏好简洁温暖风格

## 功能模块

### 一、字卡聊天

| 需求 | 说明 |
|------|------|
| 交互形式 | 类似微信聊天界面，用户输入文字发送，系统模拟对方回复 |
| 字卡类型 | 文字字卡、拍一拍、表情包（图片） |
| 字卡添加 | 对话框输入，换行分割不同字卡 |
| 回复逻辑 | 随机从字卡库抽取 |
| 回复前 | 显示"对方正在输入…"状态 |
| 回复延迟 | 设置中可调 |
| 回复条数 | 设置中可调（每次 1~N 条） |
| 消息引用 | 双方消息可引用；对方不引用 10 分钟前内容 |
| 头像 | 双方头像可由用户自行修改 |
| 气泡颜色 | 用户与对方颜色区分，米白/米黄色系 |

### 二、陪伴计时

| 需求 | 说明 |
|------|------|
| 场景 | 学习、吃饭、睡眠（固定）+ 自定义 |
| 模式 | 正计时（累计）/ 倒计时（目标时长） |
| 提醒 | 到时手机通知，震动可选 |
| 记录 | 自动保存，可查看/删除 |

### 三、每日记录

| 需求 | 说明 |
|------|------|
| 情绪标签 | 预设若干，用户自由增删 |
| 用户小纸条 | 每天 1~3 条 |
| 另一半状态 | 从标签随机取，可随时出现 |
| 另一半留言 | 每天 1 条，由 1~3 条字卡拼接，空格间隔 |
| 生理期入口 | 右上角按钮，跳转生理期子页面 |

### 四、书信

| 需求 | 说明 |
|------|------|
| 写信 | 用户自由书写 |
| 回信 | 24 小时内自动回复 |
| 回信内容 | 10~15 条字卡组成 |

### 五、生理期

| 需求 | 说明 |
|------|------|
| 记录 | 每次经期起止日期 |
| 预测 | 基于历史数据推测下次 |
| 阶段留言 | 经期/卵泡期/排卵期/黄体期/通用，共 5 类 |
| 留言管理 | 用户可自行修改（类字卡形式） |

### 六、数据管理

| 需求 | 说明 |
|------|------|
| 存储位置 | 浏览器 IndexedDB（本机） |
| 导出 | JSON 文件，按模块勾选 |
| 导入 | 选择文件，按模块勾选，合并/覆盖 |
